+ [N24News](https://github.com/billywzh717/N24News): A News Dataset for Multimodal Classification.
+ [LearningQ](https://github.com/AngusGLChen/LearningQ): A Large-scale Dataset for Educational Question Generation.
+ [Foursquare](https://sites.google.com/site/yangdingqi/home/foursquare-dataset#h.p_7rmPjnwFGIx9): A Global-scale Check-in Dataset with User Social Networks.

# <a name="code">Code</a>
